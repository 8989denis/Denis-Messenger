# Denis Messenger

Denis Messenger to program, który pozwala na wysyłanie wiadomości w konsoli, komunikując się z innymi użytkownikami na serwerze.

## Wymagania

- **Windows 10/11** w wersji komilacji 1809 lub nowszej. (Win + R i wpisać winver)
- **Node.js** w wersji 23 lub wyższej. (Start.bat lub wpisać w cmd winget install OpenJS.NodeJS)

## Instalacja

1.Wpisz w cmd winget install OpenJS.NodeJS aby zainstalować potrzebne oprogramowanie do uruchomienia lub włącz mój program "start.bat" dołączony do plików.

Jeśli chcesz mieć skrót na pulpicie pliku "start.bat" to naciśnij prawym przyciskiem na "start.bat" a następnie "Utwórz skrót" przeciągnij na pulpit.
